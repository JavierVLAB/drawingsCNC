# drawingsCNC
Drawings in P5 js for print a CNC or Plotter




# Commands for GCODE

- G90 Absolute coordinates
- G91 Relative coordinates
- G91 X20 Y-10 move 20 in X and -10 in Y
- G90; G1 F2000 X20 Y-10 go to (20,-10) at speed 2000
- M03 S0 Pen up
- M03 S100 Pen Down
